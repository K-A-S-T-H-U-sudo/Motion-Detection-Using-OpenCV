# Motion Detection Using OpenCV

## Overview
A real-time motion detection system built using Python and OpenCV. The application captures video from a webcam and detects moving objects using frame differencing techniques.

## Features
- Real-time webcam monitoring
- Motion detection
- Bounding box around moving objects
- Live status display

## Technologies Used
- Python
- OpenCV
- Imutils
- Computer Vision
